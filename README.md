# The Rabbit Hole NYC

This is the code for The Rabbit Hole's [website][].

## Requirements

- [Vagrant][]
- [VirtualBox][]

## Getting Started

```
$ git clone git@github.com:tsmall/rabbithole.git
$ cd rabbithole
$ vagrant up
$ vagrant ssh
$ cd /vagrant
$ rake
```


[vagrant]: http://vagrantup.com/
[virtualbox]: http://www.virtualbox.org/
[website]: http://rabbitholenyc.com
